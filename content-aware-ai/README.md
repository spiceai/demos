## Start app and spice docker compose

```bash
make
```

Navigate to `http://localhost:3001/thread1`

## Start web app

Add `./app/.env.local`

```
SPICE_HTTP_ENDPOINT=http://localhost:3001
```

Run
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
